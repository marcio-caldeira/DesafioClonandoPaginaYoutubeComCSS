# Desafio do BootCamp Santander 2025 Front-End DIO

## "Clonando a Página do YouTube com CSS"

### 📌 Entendendo o Desafio

Neste desafio, você colocará a mão na massa e irá clonar a página do YouTube utilizando CSS, colocando em prática todos os conceitos aprendidos, principalmente sobre **Flexbox**.
